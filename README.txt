1) Αντιγράψτε τον φάκελο project μέσα στα htdocs του XAMPP.

2) Ανοίξτε το XAMPP και τρέξτε τον Apache και την MySQL.

3) Ανοίξτε το project API με το intellij ελέγξτε τις ρυθμίσεις στο application.properties
για την βάση σας να είναι σωστό το όνομα και οι κωδικοί,(για εμάς ήταν eshop το όνομα της βάσης
και το όνομα και password root)  και τρέξτε το application.

4) Εφόσον τα έχετε τώρα όλα έτοιμα δεν μένει παρα να εισάγετε στην βάση μερικά προϊόντα τα οποία θα βρείτε
στο phones.json αρχείο. Ανοίγεται λοιπόν το postman και περνάτε ένα ένα όσα products θέλετε με POST request
στο localhost:8080/api/v1/products .

5) Ανοίγετε το index της σελίδας μπαίνοντας εδώ απο τον browser σας --> http://localhost/project/index.php 

6) Όλα πρέπει να δουλεύουν όπως πρέπει και λογικά μπορείτε να δουλέψετε κανονικά τη σελίδα.
